# Usecases Library

Static demo showing 702 generated usecases with search, filters and cards.

Open `index.html` in a browser (no server required) to view, or run a local server for localhost access.

Run server (optional):

```
# Python 3
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

The use cases are loaded from `usecases_101.json`. To use your own data, replace the content of `usecases_101.json` with your JSON array of usecases following the same structure.
